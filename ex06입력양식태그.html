Form 태그

  • form 태그는 입력 양식 태그
  • 사용자에게 데이터를 입력받아서 전송할때 사용하는 태그
  • 필수 조건 : action, name, submit
    1. action : 데이터를 전송할 위치
    2. name : 데이터의 이름
    3. submit : 데이터를 보낼 시점
